This serves as a prototype for a real autonomous search and rescue operation to eliminate the potential threats to search and rescue teams' lives. The system is designed to scan an area with a camera mounted on a quad copter to search for a target object. The quad copter will then send the location of this object back to a ground rover which will navigate to the object.{% endblock %} {% block img1 %}{% endblock %} {% block img2 %}{% endblock %} {% block status %} We are currently midway through the project.
